Ecommerce - Final Project

Used tools:

  • Front-end: React, Redux
  • Back-end: NodeJS, Express
  • Database - ORM: Sequelize, PostgreSQL
  • CSS Styles: MaterialUI
  • Autentication: Auth0
  • Payment gateway: MercadoPago

Project Information

  • Category: Full Stack Aplication
  • Client: Bootcamp Henry
  • Date: ene, 2023
  • Project URL: https://pf-ecommerce-peach.vercel.app

Application for Ecommerce in which you can manage products by categories and from it power, among other functions:

  • - Complete list of products (catalogue)
  • - Search Bar to quickly find products
  • - Details of an individual product
  • - Being able to create and edit products
  • - Being able to add or remove categories
  • - Filter by categories.
  • - Manage item availability (stock)
  • - Being able to leave product reviews
  • - Being able to add or remove items to my cart
  • - Being able to buy all the items in a my cart. (check out)
  • - Specify a shipping address and an email when I checkout
  • - Receive a purchase confirmation email
  • - View a list of all orders, so you can view and review orders
  • - View the details of a specific order
  • - Being able to change the status of an order (created, processing, cancelled, complete)
  • - Being able to pay for an order using a payment gateway (in: MercadoPago, Stripe, etc.).
  • - Being able to create, modify, inactivate user accounts, or turn them into managed
  • - Being able to log in with users of social networks.
  • - Being able to add, see and remove products from my wishlist.
